Infiniboot
==========

Infiniboot is a Bootstrap theme that looks kind of like Infinidat products.

It should work on top of the stock [Bootstrap v4][1].

[1]: https://github.com/twbs/bootstrap/tree/v4-dev


Dependencies
------------

    npm i -g grunt-cli


Getting started
---------------

    npm i
    grunt
    open http://localhost:3000
